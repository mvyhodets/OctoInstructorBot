OctoInstructorBot

Бот для личного кабинета сотрудников компании Octoskills.

Цель бота - обеспечить удобный интерфейс для работы тренеров и менеджеров в Telegram.
Технологии:

    - Python 3.10
    - Aiogram 3.0

API

    - Telegram Bot API
    - 1C:Фитнесс API версия 3

Контейнеризация

    - Docker
